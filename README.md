# Fund-Selection-Tool-V.1-31-03-2026

# 📊 Fund Selection Tool (V.1) 🇹🇭
เครื่องมือช่วยวิเคราะห์และคัดเลือกกองทุนรวม (Beta Version) เพื่อการตัดสินใจลงทุนอย่างมีระบบ

---

## 🚀 ขั้นตอนการใช้งานเบื้องต้น

### 1. การเลือกกองทุน (Fund Selection)
เริ่มต้นที่หน้าแรกเพื่อค้นหากองทุนที่ตรงกับสไตล์การลงทุนของคุณ:
* **ตัวกรองอัจฉริยะ:** คุณสามารถเลือกกรองตาม **หมวดหมู่ (Category)**, **ประเภทกองทุน (Type)**, **สไตล์การลงทุน**, และ **นโยบายเงินปันผล**
* **ระบบ Checkbox:** ติ๊กเลือกกองทุนที่สนใจที่ช่องด้านหน้า เพื่อส่งรายชื่อกองทุนเหล่านั้นเข้าสู่กระบวนการคัดสรรเชิงลึกในขั้นตอนถัดไป
* **จุดไฮไลท์:** ระบบจะมีการ **มาร์คสี (Color Highlight)** เพื่อช่วยให้กองทุนที่คุณสนใจโดดเด่นและสังเกตง่ายเมื่อสลับไปดูในเมนูวิเคราะห์อื่นๆ

### 2. การวิเคราะห์ผลตอบแทน (Performance Analysis)
ในเมนู **Annualized Return** หรือ **Calendar Year**:
* **Sortable Table:** คุณสามารถ **คลิกที่หัวตาราง (Table Header)** เพื่อเรียงลำดับข้อมูลผลตอบแทนจาก **มากไปน้อย** หรือ **น้อยไปมาก** ได้ทันที ช่วยให้เห็นกองทุนที่ทำผลงานได้โดดเด่นที่สุดในแต่ละช่วงเวลา

### 3. การตรวจสอบค่าธรรมเนียม (Fees)
* **Fee Structure:** แสดงโครงสร้างค่าธรรมเนียมที่เหมาะสมเพื่อเปรียบเทียบความคุ้มค่า
* *หมายเหตุ: ปัจจุบันยังเป็นข้อมูลจากฐานข้อมูลชุดทดสอบ (Static Data) ยังไม่ใช่การดึงข้อมูลแบบ Real-time*

### 4. สัดส่วนการถือครอง (Top 10 Holdings)
* **Master Fund Insights:** ดูรายละเอียดหุ้นหรือสินทรัพย์ 10 อันดับแรกที่กองทุนถือครอง เพื่อวิเคราะห์ความเสี่ยงและแนวทางของ Master Fund
* *หมายเหตุ: ข้อมูลในส่วนนี้เป็นไฟล์ข้อมูลชุดปัจจุบัน (Static Data) เช่นเดียวกับส่วนของค่าธรรมเนียม*

---

## 🛠 ข้อมูลทางเทคนิค (Internal Logic)
* **Color Ranking:** ระบบใช้การแบ่งกลุ่มผลงานออกเป็น 6 ระดับ (Sextiles) โดยใช้สีตั้งแต่ `--rank-1` (เขียวเข้ม) ไปจนถึง `--rank-6` (แดง) เพื่อการเปรียบเทียบที่รวดเร็ว
* **Data Driven:** พัฒนาด้วย HTML/CSS และ JavaScript โดยดึงข้อมูลหลักจาก `data.js`

---
*คำเตือน: เครื่องมือนี้จัดทำขึ้นเพื่อช่วยอำนวยความสะดวกในการวิเคราะห์ข้อมูลเท่านั้น ผู้ลงทุนควรศึกษาข้อมูลเพิ่มเติมจากหนังสือชี้ชวนก่อนตัดสินใจลงทุน*
