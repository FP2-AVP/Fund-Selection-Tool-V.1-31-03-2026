## 🚀 ขั้นตอนการใช้งานเบื้องต้น

### 1. การเลือกกองทุน (Fund Selection)
เริ่มต้นที่หน้าแรกเพื่อค้นหากองทุนที่ตรงกับสไตล์การลงทุนของคุณ:
* **ตัวกรองอัจฉริยะ:** สามารถเลือกกรองตาม **หมวดหมู่ (Category)**, **ประเภทกองทุน (Type)**, **สไตล์การลงทุน**, และ **นโยบายเงินปันผล**
* **ระบบ Checkbox:** ติ๊กเลือกกองทุนที่สนใจที่ช่องด้านหน้า เพื่อส่งรายชื่อกองทุนเหล่านั้นเข้าสู่กระบวนการคัดสรรเชิงลึกในขั้นตอนถัดไป
* **จุดไฮไลท์:** ระบบจะมีการ **มาร์คสี (Color Highlight)** เพื่อให้กองทุนที่คุณเลือกมีความโดดเด่นและสังเกตง่ายในขั้นตอนถัดๆ ไป

### 2. การวิเคราะห์ผลตอบแทน (Performance Analysis)
ในเมนู **Annualized Return** หรือ **Calendar Year**:
* **Sortable Table:** สามารถ **คลิกที่หัวตาราง (Table Header)** เพื่อเรียงลำดับข้อมูลผลตอบแทนจาก **มากไปน้อย** หรือ **น้อยไปมาก** ได้ทันที ช่วยให้เปรียบเทียบผลงานในแต่ละช่วงเวลาได้ชัดเจน

### 3. ค่าธรรมเนียมเหมาะสม (Fees)
* **Fee Structure:** แสดงโครงสร้างค่าธรรมเนียมเพื่อเปรียบเทียบความคุ้มค่า
* *หมายเหตุ: ปัจจุบันยังเป็นไฟล์ข้อมูล (Static Data) ยังไม่ใช่ระบบ Real-time*

### 4. สัดส่วนการถือครอง (Top 10 Holdings)
* **การโหลดข้อมูล:** ในหน้าพื้นฐาน จำเป็นต้องกด **"โหลดทั้งหมด"** ก่อนทุกครั้ง เพื่อให้ระบบดึงข้อมูลให้พร้อมใช้งาน
* **การเปรียบเทียบ:** เมื่อข้อมูลแสดงผลแล้ว คุณสามารถติ๊กเลือกกองทุนเพื่อดูสัดส่วนการลงทุน (Master Fund Insights) และเปรียบเทียบกันได้
* *หมายเหตุ: ข้อมูลในส่วนนี้ยังเป็นไฟล์ข้อมูลคงที่ (Static Data)*

---

## 🛠 ข้อมูลทางเทคนิค (Internal Logic)
* **Color Ranking:** ระบบใช้การแบ่งกลุ่มผลงานออกเป็น 6 ระดับ (Sextiles) ตั้งแต่ `--rank-1` (เขียวเข้ม) ถึง `--rank-6` (แดง)
* **Data Driven:** พัฒนาด้วย HTML/CSS และ JavaScript โดยใช้ `data.js` เป็นฐานข้อมูลหลัก
